lbdp {phylopomp}R Documentation

Linear birth-death process.

Description

Run the LBDP simulator.

Usage

playLBDP(data = NULL, ..., lambda, mu, n0, psi, t0 = 0, times, tree = FALSE)

Arguments

data

optional data frame; output from playLBDP.

...

additional arguments; ignored.

lambda

birth rate

mu

death rate

n0

initial population size

psi

sampling rate

t0

initial time

times

times at which output is requested.

tree

logical; represent the genealogical tree in Newick format?

Value

A tibble with state attribute.

See Also

Other Genealogy processes: moran, sirws

Examples

playLBDP(lambda=1,mu=0.5,psi=1,n0=3,times=seq(0,4,by=0.2),tree=TRUE) -> x
plot(x,points=TRUE)

y <- getInfo(x,prune=FALSE)
plot(y,points=TRUE)

library(ggplot2)
y$lineages %>%
  ggplot(aes(x=time,y=lineages))+
  geom_step()+
  geom_vline(xintercept=y$etimes,alpha=0.1)

y$cumhaz %>%
  ggplot(aes(x=exp(-Lambda)))+
  stat_ecdf()+
  geom_abline(slope=1)
