moran {phylopomp}R Documentation

Moran genealogy process.

Description

Run the MGP simulator.

Usage

playMoran(
  data = NULL,
  ...,
  n,
  mu,
  t0 = 0,
  times,
  tree = FALSE,
  stationary = TRUE
)

Arguments

data

optional data frame; output from playMoran.

...

additional arguments; ignored.

n

population size

mu

Moran event rate

t0

initial time

times

times at which output is requested.

tree

logical; represent the genealogical tree in Newick format?

stationary

logical; should the initial genealogy be drawn from the stationary distribution?

Value

A tibble with state attribute.

See Also

Other Genealogy processes: lbdp, sirws

Examples

playMoran(n=5,mu=5,times=1:10,t0=0,tree=TRUE) -> x
playMoran(x,times=11:20,tree=TRUE) -> x
plot(x)

playMoran(n=20,mu=20,times=1:20,stationary=FALSE,tree=TRUE) -> x
plot(x)

y <- getInfo(x,prune=FALSE)
plot(y,points=TRUE)
